Skillset

This section lists my personal Web Dev skills for both front and back end development, which I'm always looking to expand and improve on.

HTML5 & CSS3

The most basic, and yet the most fundamental tools of web development. I've learned the importance of component layout, as well as size translation for different sized screens.

Javascript

Vanilla Javascript is a very important tool, and can be scaled up to deliver high levels of functionality. Of the programming languages I'm familiar with, this is most definitely my favourite.

jQuery

jQuery is a helpful tool to reduce the amounts of code required when building applications. I've found it especially useful for dynamic pages where elements need to be manipulated from the DOM, and when collecting and checking information from input fields.

Express.js

Express API routing is quite useful when working in more complex websites and applications. When sending data to and from databases, its my preferred method of building out pathways.

Node.js

Node is the best package manager out there, and is incredibly helpful when importing libraries and packages as needed for any projects. Node modules grant high levels of functionality and even access to cross-language coding.

React

React provides a highly versatile framework and structure for both building page components and deploying them. The components also allow for much more complex single-page applications, such as the Fortnite 2D game I helped build.

SQL

SQLite, Postrgresql and MySQL are all highly useful database languages and allow for much more nuanced database querying. SQLite was the database language I used in both the Fortnite 2D and Expense Tracker apps, and was particularly useful for the latter.

MongoDB

MongoDB has less functionality than SQL and its derivatives, but in turn it takes less time to perform large amounts of queries and can scale up more dramatically.
