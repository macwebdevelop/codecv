This is a game I built during an online course using HTML, CSS and Javascript, where players attempt to guess the colour given by the rgb code at the top of the screen. The game has different difficulties which add more squares to pick from, and the game gives randomized colours to each square on startup, on pressing 'New Colours', and finally on changing difficulty.

Fortnite 2-D

This game was a university project which our class built in teams of two. I learned about using NodeJS to install modules, and worked with Express, React and jQuery, with a SQLite3 Database to manage profiles. The game can be played with multiple players when run on a server, and had mechanics like health and ammunition refilling, indestructible and destructible obstacles, border walls, and even buildings with roofs which disappear once the player is underneath them. My main role was to build the CRUD operations, using the express app to interact with our SQLite3 database, and to expand the React components to build update, register, and delete pages.

UpGrade

This application is both web and android based, and was a group project for a software development class. This site was built over four sprints of two weeks, in a group of six students. We also made use of Jira to keep track of our respective user stories, and worked to use INVEST principles to maintain our development momentum across the sprints. I proposed the idea of a tool which could aid students in keeping track of their assignments and test scores, as our school system does not always require professors to upload the grades for the students in their courses if they give them a physical copy of it. I learned basic app development using Android Studio, including using grid layouts to build each activity screen. I also got some exposure to Firebase as I had to build a function for implementation in the app that was already present in the website.
